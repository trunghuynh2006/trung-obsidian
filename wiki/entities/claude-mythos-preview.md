# Claude Mythos Preview

**Type:** product
**Also known as:** Mythos Preview, Claude Mythos

## Overview

Claude Mythos Preview appears in the wiki as an Anthropic model reportedly strong enough at code generation and vulnerability discovery that the company chose not to release it broadly. In the current source set, its significance is not benchmark performance in general, but the claim that it can uncover security flaws across extremely widely used software systems at unusual scale and speed.

The current evidence is narrow and should be treated cautiously. The claims about Mythos come primarily from Anthropic's own announcement and an opinion-driven article that builds strategic conclusions on top of those claims. Still, the model matters because it sharpens an emerging pattern already visible in the wiki: coding capability, cyber capability, and governance pressure are converging.

## Key Facts

- Anthropic reportedly released Mythos Preview only to a limited consortium of about 40 companies and critical-infrastructure operators [[wiki/sources/anthropic-ai-claude-mythos]]
- The source says Mythos found thousands of high-severity vulnerabilities, including some in every major operating system and web browser [[wiki/sources/anthropic-ai-claude-mythos]]
- Anthropic reportedly does not plan to make the preview model generally available [[wiki/sources/anthropic-ai-claude-mythos]]
- Project Glasswing is the name given to the restricted consortium built around the model's defensive use [[wiki/sources/anthropic-ai-claude-mythos]]

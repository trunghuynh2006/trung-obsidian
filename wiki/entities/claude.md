# Claude

**Type:** product
**Also known as:** Claude.ai, Anthropic Claude

## Overview

Claude is [[wiki/entities/anthropic]]'s conversational AI product family. In this wiki's current source set, Claude appears both as a model/interface for everyday reasoning and as the product context around more specialized tools such as Claude Code and Claude Mythos Preview.

The Mark Cuban source adds a practical learning role: Claude can be used as an adaptive tutor for learning how to build business agents. Cuban's suggested prompts ask Claude to teach agent-building for small businesses, create question-based study guides, correct the learner, and adapt to their current knowledge level.

## Key Facts

- Recommended by Mark Cuban as a starting point for learning how to create agents for small businesses. [[wiki/sources/mark-cuban-future-proof-career-ai-2026-4]]
- Can generate learning paths, study guides, and adaptive feedback loops for agent-building skill acquisition. [[wiki/sources/mark-cuban-future-proof-career-ai-2026-4]]
- In Business Insider's test, Claude suggested practical business-agent targets and orchestration tools such as LangGraph, CrewAI, and AutoGen. [[wiki/sources/mark-cuban-future-proof-career-ai-2026-4]]

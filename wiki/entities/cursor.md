# Cursor

**Type:** product / company
**Category:** AI coding tool

## Overview

An AI-powered code editor built on VS Code, developed by Anysphere. One of the earliest and most widely adopted AI coding tools, Cursor provides AI autocomplete, chat-based code generation, and increasingly agentic code-writing capabilities. Central to the "code overload" story of 2026.

## Key Events

- Early adoption drove dramatic productivity gains — e.g., one financial services firm 10× their monthly code output after adopting Cursor. [[wiki/sources/ai-code-overload]]
- **December 2025**: Acquired **Graphite**, a code-reviewing bot startup — a direct response to the review bottleneck that Cursor's own productivity gains helped create.
- **2026**: Building code-review and prioritization products to help engineers manage AI-generated code volume.

## Notable Quotes

> "The software development factory kind of broke. We're trying to rearrange the parts." — Tido Carriero, Cursor head of engineering [[wiki/sources/ai-code-overload]]
